# 📊 HR Payroll MIS Dashboard (Excel + SQL Project)

## 🚀 Project Overview
This project is an end-to-end HR Payroll MIS system built using SQL and Excel. It analyzes employee data, payroll structure, attendance, and leave patterns to generate business insights through a dashboard.

---

## 🛠️ Tech Stack
- PostgreSQL (Data handling & queries)
- Microsoft Excel (Dashboard & Pivot Tables)
- SQL (Joins, Aggregations, Group By)
- GitHub (Version Control)

---

## 📁 Dataset Modules

### 1. Employees Table
- Employee ID
- Name
- Department
- Joining Date
- Salary

### 2. Payroll Table
- Basic Salary
- Bonus
- Deductions

### 3. Attendance Table
- Employee ID
- Present Days
- Absents

### 4. Leaves Table
- Leave Type
- Leave Days

---

## 📊 Key KPIs
- Total Employees
- Total Salary Expense
- Average Salary
- Total Attendance Days
- Total Leave Days

---

## 📈 Dashboard Visuals
- Department-wise Salary Distribution (Bar Chart)
- Employee Count by Department (Column Chart)
- Leave Analysis (Pie Chart)
- Attendance Summary (Bar Chart)
- Salary Distribution Trend

---

## 🧠 SQL Features Used
- SELECT
- SUM, COUNT, AVG
- GROUP BY
- CASE WHEN
- Joins (Employee + Payroll + Attendance + Leaves)

---

## 📤 Workflow
1. Cleaned raw data using SQL queries
2. Exported query results as CSV
3. Imported data into Excel
4. Created Pivot Tables
5. Built interactive dashboard with charts

---

## 📌 Key Insights
- Identified high salary departments
- Analyzed employee attendance patterns
- Evaluated leave distribution across employees
- Compared salary structure across departments
